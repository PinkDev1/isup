# Check Host Faster Than Curl

## isup.sh
isup.sh is a simple bash tool made for checking alive hosts, subdomains from a dumped files , bruteforced subdomains .

**USAGE**
  ```bash
  ./isup.sh filename
  ./isup.sh filepath
  ```
  **HELP**
  ```bash
  ./isup.sh -h
  ./isup.sh --help
  ```
